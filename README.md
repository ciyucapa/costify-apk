# 📱 Costify

Costify es una aplicación móvil desarrollada con React Native enfocada en pequeños negocios gastronómicos y emprendimientos que necesitan controlar costos, inventario y rentabilidad.

## ✨ Funcionalidades

- 📦 Gestión de inventario
- 💰 Control de ganancias
- 🏭 Producción sugerida
- 📈 Ventas y métricas
- 🛒 Lista inteligente de compras
- 🔔 Alertas de stock
- 🔐 Autenticación Firebase

## 🧰 Tecnologías

- React Native
- Expo
- Firebase Authentication
- Firebase Firestore
